

TIENDA ONLINE
Mi proyecto se basa en una tienda online con un carrito de compra que ofrece productos como equipos de sonidos consolas, micrófonos, luces, accesorios varios, usb, conectores y cables para aficionados de la musica, djs amateur y profesionales. El usuario podra consultar mediante el formulario de contacto, buscar mediante el buscador, podra filtrar por categoria, agregar al carrito y por ultimo comprar si asi lo desea.
